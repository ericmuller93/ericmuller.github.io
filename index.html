My Experience in Python!

I have built a few applications in python, mainly using pycharm. You can find them on my Github.
The first is an application that will let you select two directories and move files of a specific extension from the first to the second. It then stores what file you moved and the last time it was edited in a SQLite 3 database. I created the GUI using tkinter.
Another is a fictional website for a hotel with a food ordering system. I utilized Django to create this.There is an admin function to create, update and delete food items from the menu.
Last and certainly not least, I worked with a SCRUM team to develop a web scraper in Django. We used modules like BeautifulSoup and Selenium to pull data from various websites and have this data in one central place. The user could then see a personalized feed of the weather, upcoming events, and applicable news story on their site.

My Experience in C#!

C# was an exciting language to learn. I very much enjoyed working with MVC in Visual Studio to make dynamic websites. You can find a car insurance quote generator on my Github. This project really helped me utilize the power of MVC. When a user enters all of their information, the system will calculate their quote and direct them to a details page. On the back end all of this information is stored in a SQL server database. An admin view was created for a site owner to maintain all of the entries.
In addition to MVC I made many console applications. One of my favorites is a TwentyOne or BlackJack game. A user specifies how much money they brought and proceeds to play and bet until they run out. This application helped me build a very solid understanding of not only C# but Object Oriented Programming too. I feel it's very important to understand concepts like inheritance and exception handling to make a proper program.
